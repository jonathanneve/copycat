TCcLog.FieldByName Method
TCcLog Class | See Also

Retieve field object by name.

function FieldByName(cFieldName: string): TCcField; override;
TCcField FieldByName(string cFieldName) override;
Description

Use field name to access the TField object, in order to read or write the value of a field. 

Please note however, that application shouldn't normally change the values of any fields of the replication log, and doing so may cause unpredicted misbehavior.

TCcReplicator.LogErrors Property
TCcReplicator Class
Show All

Set LogErrors to indicate whether replication errors should get automatically logged to the RPL$ERRORS table in the local database.

property LogErrors: Boolean;
__property Boolean LogErrors;
Description

LogErrors is set to False by default for backwards compatibility reasons.

Notes

If you are upgrading from a version older than 3.01.0, you must first reconfigure your database (using the TCcConfig component) so that the RPL$ERRORS table will get created.

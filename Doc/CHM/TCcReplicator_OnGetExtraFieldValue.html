TCcReplicator.OnGetExtraFieldValue Event
TCcReplicator Class
Show All

This event fires during replication if ExtraFields are defined.

property OnGetExtraFieldValue: TCcFieldValueEvent;
__property TCcFieldValueEvent OnGetExtraFieldValue;
Description

ExtraFields are fields that are missing in one of the two nodes and that therefore cannot be replicated normally, but if you implement this event, you can supply a value for them to be used when inserting or updating the destination database.

TCcReplicator.Closing Property
TCcReplicator Class
Show All

Indicates that the replication in the process of closing.

property Closing: Boolean;
__property Boolean Closing;
Description

Closing is set to true whenever the user chooses to abort the replication process (whether by the AbortReplication method, or in response to an OnReplicationError event). As soon as an appropriate moment is found, replication then stops, and Closing is set back to false.

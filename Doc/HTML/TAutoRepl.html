CcReplicator

Description

Use TAutoRepl to automatically launch a replication session periodically, if there is none already in progress. 

 

To enable automatic replication, set Enabled to True, and call the Start method when you are ready to start the timer. Automatic replication can be temporarily disabled at any time, by calling the Stop method.

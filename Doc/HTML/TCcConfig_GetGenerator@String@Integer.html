Build SQL statement for synchronizing a primary key or generator field, based on a generator.

function GetGenerator(GenName: String; Increment: Integer): String;
String GetGenerator(String GenName, int Increment);
Parameters
Parameters 
Description 
GenName: String 
Name of the generator to use. 
Increment: Integer 
Amount to increment the generator by. 
Returns

The generated SQL statement, in a format suitable to be put in a 'PKn_GEN' or 'GENn_VALUE' field of Tables.

Notes

This function does not alter the Script property, it merely returns the SQL statement.

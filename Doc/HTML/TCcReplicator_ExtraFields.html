TCcReplicator.ExtraFields Property
TCcReplicator Class | See Also
Show All

Use ExtraFields to hold the list of fields which are needed in the destination database but do not exist in the source database.

property ExtraFields: TStringList;
__property TStringList ExtraFields;
Description

ExtraFields is cleared before each new row, so you can use OnRowReplicating to fill it. The OnGetExtraFieldValues event is fired in order to get the values for the extra fields.

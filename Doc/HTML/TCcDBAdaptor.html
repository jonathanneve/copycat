TCcDBAdaptor Class
CcProviders | Properties | Methods | Legend
Show All

TCcDBAdaptor is the abstract ancestor class for the adaptor classes providing database-specific features for each of the supported database types.

Class Hierarchy

Description

You can access the DBAdaptor of your database connection by using the TCcConnection.DBAdaptor property.

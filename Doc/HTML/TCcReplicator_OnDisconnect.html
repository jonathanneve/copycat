TCcReplicator.OnDisconnect Event
TCcReplicator Class | See Also
Show All

Fired before disconnecting from both local and remote database.

property OnDisconnect: TCcNotifyEvent;
__property TCcNotifyEvent OnDisconnect;
Description

Note that this event is not fired when the connection is unintentionally lost, but only if the application calls the Disconnect method, or if KeepConnection is set to False.
